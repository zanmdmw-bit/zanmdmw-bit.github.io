集市开门前，税吏先检查秤砣，行会的人再检查摊位，守卫最后收走入城费。等第一枚铜币真正落到商贩手里，它已经背负了农田的收成、领主的权利、道路的危险和许多双看不见的手。

双陆世界拥有跨国通行的币制，却不是一个人人靠工资生活的现代市场。农民用谷物交租，用劳役偿还义务；城市工匠以银币计价，却受行会、税卡和原料商控制；贵族用金币购买盔甲、土地与忠诚，大商人则让财富以汇票、债权和仓单的形式越过国境。

## 一套跨越国境的金银铜

大陆主要政权采用三层公衡币制：**1 金币等于 100 银币，1 银币等于 100 铜币，因此 1 金币等于 10,000 铜币。**各国可以在钱币正面铸皇帝、王冠、舰船或神徽，但只要重量、纯度、尺寸和防伪印记符合大陆公衡标准，商人就按同一面值接受。

这种统一并非源于诸国互相信任。黑日战争时期，古代五国、三个矮人王国、九大神殿和商人组织为了结算军饷、粮草与跨国补给建立了公衡制度；今天则由大陆公衡院、神殿金库、矮人验金师和黄金道路商盟共同维持。正常换币手续费约为 0.5%—3%，钱币被削边、磨损、纯度可疑或来自交战国时会被称重折价，灾荒与信用崩溃地区的手续费可能升到 5%—15%。

铜币负责一碗粥、一块面包和城门小税；银币负责工资、衣服、工具、住宿和普通武器；金币则进入房屋、土地、奴隶、战马、魔法物品、商队货物和大型委托。对一年只有几枚金币现金收入的家庭而言，一枚金币不是“较大的零钱”，而是一段可能改变婚姻、债务和生死的财富。

## 钱并没有进入每一个角落

农村家庭的大部分收入和支出并不经过钱袋。粮租、劳役、柴草、放牧权、共同修渠、领主磨坊和神殿什一捐都可以用实物或工作日结算。一个家庭可能仓里有粮、圈里有鸡，却拿不出医师要求的三十银币；也可能在账面上欠了领主一年地租，却仍靠邻里交换熬过冬天。

硬币因此经常被埋藏、缝进衣服、藏在墙洞或交给神殿保管。普通人储蓄不是为了投资回报，而是为了下一次歉收、征兵、疾病和婚丧。货币越稳定，领主越容易把原本以实物缴纳的义务折成现金；这既方便贸易，也把没有现金收入的农户推向借贷。

## 普通人的生活尺度

在治安正常、没有灾荒的普通人类城镇，一个成年人每天购买基本食物约需 **30—40 铜币**。自行做饭的贫民可以压到 18—30 铜币，重体力劳动者通常需要 40—70 铜币；一名没有身份和工具的异乡短工半日可能只得 5—15 铜币，稳定普通工人则约为每日 25—50 铜币并附一餐。

| 生活与劳动项目 | 常见基准 |
| --- | ---: |
| 黑麦硬面包 | 7—10 铜 |
| 豆肉炖菜 | 8—18 铜 |
| 旅店通铺 | 20—40 铜／晚 |
| 贫民区狭小单间 | 6—20 银／月 |
| 城镇守卫 | 12—30 银／月，常另有伙食或装备 |
| 熟练木匠、铁匠、裁缝 | 80 铜—3 银／日 |
| 商队护卫 | 20 银—1 金／月 |
| 正式低阶法师 | 2—10 金／月 |

普通城镇家庭一年的现金收入通常只有 **1—4 金币**，熟练工匠家庭约为 5—15 金币。这个数字不能简单拿每日工资相乘：冬季停工、无薪劳役、工具损耗、疾病、行会会费和雇主拖欠都会吞掉收入。农村家庭自产的粮食也有土地、种子、储存、磨坊与纳税成本，只是这些成本不总以硬币出现。

## 税不是一张单独的账单

平民很少只交一种税。领主可能征收地租、收获分成、磨坊费、烤炉费、桥税、道路税、城门税、集市摊位费、牲畜登记费、婚姻许可费和继承承认费；神殿收取什一捐或节庆供奉；国家在战争时追加军费、征粮与马匹。某一项税看似不高，叠加后却足以决定家庭能否留下种粮。

税负也不只落在穷人身上，但贵族、神殿和特许商号更容易用免税状、包税权和政治关系转移成本。大商人把关税写进售价，领主把军费摊到村庄，最后承担风险的常是议价能力最低的人。

征税并不总是高效。道路遥远、文书不全和地方腐败会让中央只能把某地税收承包给贵族或商人。包税人预先向王室交钱，再用自己的卫兵向居民追讨；从国库角度看，这带来了确定收入，从居民角度看，国家只是把强制权卖给了最擅长收债的人。

## 行会控制城市的手艺与机会

行会规定谁能当学徒、学多少年、使用什么材料、在哪条街开店以及商品最低质量。它会替成员追债、照顾遗孀、主持葬礼、组织节庆和向领主谈判，也会没收无照工具、排斥外乡人、限制新作坊和操纵价格。

因此行会既是工人保护，也是垄断。成为正式师傅意味着拥有身份、信用和参加城市政治的入口；没钱交会费、没有师承或来自异界的人，即使技术更好，也可能只能替持照者做低价黑工。魔法工坊和药剂行会尤其严厉，因为一次失败不仅损失商品，还可能烧毁整条街或释放污染。

## 商路把国家缝合起来

阿尔赛恩输出粮食、亚麻、纸张、葡萄酒和大量官营铁器；瓦尔卡恩输出铁、魔铁、盔甲与矿业机械；米尔瑟拉出售魔晶、法具、药剂、符文材料和教育；塞勒尼斯依靠盐、鱼干、香料、糖、染料、造船与海运服务获利；诺德维亚则把木材、毛皮、巨兽骨、冰系材料和耐寒坐骑送往南方。

精灵王庭出售稀有药材、精灵丝和生命魔法材料，却严格限制古树种子与高阶法术；兽人诸国输出战马、牲畜、毛皮和草原药材；三个矮人王国以高纯度金属、合金、导魔构件和可靠工具换取地表粮食。魔界走私品——梦境香料、绯月丝、黑海珍珠、契约物与禁术材料——利润最高，背后的诅咒、追踪和政治风险也最昂贵。

所谓大陆基准价只是商人共同理解的尺度，不是所有地方同价。本地产区常为基准的 70%—100%，相邻领地约 100%—130%，跨国约 120%—160%，跨越数国可达 150%—300%；大陆另一端的稀缺货物可能是产地价的十倍以上。

## 差价不能自动变成利润

一辆马车从产地走到买家手中，要支付采购定金、仓储、关税、门税、桥税、护卫、牲畜饲料、车轴损耗、腐烂、盗匪、魔物和腐败官员。跨越多个国家后，总成本增加 30%—200%十分常见。战争时某条旧路突然封闭，整支商队可能只能把货低价卖给最近的城镇。

真正赚钱的商人依靠稳定路线、本地关系、大宗采购、低损耗货物、可靠情报、垄断许可证和回程货物。一个外来者看见两地价格差一倍，并不等于他有能力让货物活着走完中间那条路。

## 信用比装满金币的箱子更轻

超过五十金币的交易通常不会搬运整箱硬币。商会汇票、神殿存金凭证、银行契书、金条、魔晶抵押、仓单、分期交割与贵族债券共同承担大额结算。黄金道路商盟的真正力量，不只是拥有商队，而是能决定哪一张纸在数千公里外仍然被当作钱。

信用也制造新的控制。贵族用未来税收抵押战争，城市用港税修筑城墙，工坊主拿下一季产量换原料。债权人不必占领土地，只要让债务到期，就能取得矿权、码头、婚约甚至任命官员的影响力。塞勒尼斯尤其擅长这种统治：舰队保护航路，保险分散风险，债务则让不属于帝国的港口也服从帝国利益。

## 战争是最大的买家

战争首先拉高粮食、铁、盐、马匹、药剂和运输价格，然后抽走田里的劳动力。军队经过时，即使留下征购凭证，农户也未必等得到兑现。贵族承包军需、商人囤积粮食、雇佣兵抬高工资，城市穷人却要在同一枚银币可以买到越来越少面包时承受后果。

长期战争也会创造繁荣地区：瓦尔卡恩的兵工城、塞勒尼斯的船坞、米尔瑟拉的战地药剂工坊都可能在别处荒芜时扩张。所谓战争经济不是所有人一起贫穷，而是财富沿着军令和合同快速转移。

## 奴隶市场的价格只属于经济篇

奴隶制度的法律身份、来源与社会后果收录于[社会与法律](world.html#society)；这里仅记录市场如何估价。奴隶价格不代表一个人的人格价值，只代表买家预期能从其剩余寿命、技能、健康、服从成本与转售风险中取得多少利益。

| 奴隶类型 | 常见成交价 |
| --- | ---: |
| 伤病、年老或高风险劳工 | 5—15 金 |
| 普通田地、矿山或苦力奴隶 | 10—25 金 |
| 健康成年家务奴隶 | 20—45 金 |
| 年轻健康、适合长期家务者 | 25—60 金 |
| 受训厨师、女仆或护卫 | 40—120 金 |
| 熟练工匠 | 60—200 金 |
| 识字管家、账房或书记 | 80—300 金 |
| 低阶法师或特殊血脉 | 200 金以上 |

艾琳 33 金币、诺拉 25 金币，均处在既定普通市场区间。成交之外还要支付身体检查、契书、登记、运输、经纪佣金和限制法具费用；跨国买卖还可能增加出口许可、担保金和贿赂。

精灵等长生稀有种通常为 300—1,500 金币，王庭血统或掌握自然魔法者可达数千乃至上万。健康魅魔幼体通常为 500—1,500 金币，能力稳定或受过礼仪、情报训练者可达 1,000—6,000 金币，特殊贵族与魔王血脉往往不公开定价。低于 300 金币的所谓魅魔常意味着诈骗、重伤、契约漏洞或足以杀死买家的政治追索。

市场并不真正“统一价格”，统一的是估价口径。战争结束后战俘大量涌入会压低苦力价格，矿难和大工程会抬高壮年劳工需求，某国禁运则会把稀有种交易推入地下拍卖。丝绒锁链商会经营高端稀有人口，红印经纪联合体处理更广泛的登记与转售；二者掌握信息、契书和买家名单，因此赚取的并不只是人口差价，更是合法性与保密费。

## 魔法没有取消稀缺

低阶魔法能够净水、保鲜、驱虫和提高局部产量，却需要施法者、材料、法具维护与许可。高级治疗可挽救重伤，价格却可能从十几金币升到数百、数千金币；断肢重生和强力复原对普通家庭而言仍等同于庄园级财富。

魔法因此没有让经济进入无限丰裕，反而创造了新的稀缺：魔晶产地、法术专利、受训法师、导魔金属和可靠的维修网络。米尔瑟拉的繁荣建立在知识许可之上，瓦尔卡恩的军力依赖矿山，神殿的治疗能力也依赖捐赠、地产和信仰义务。

## 异界知识进入旧秩序之后

地球来客可能知道更高效的作物轮作、卫生、机械、会计与组织方法，却很少拥有立刻实现它们所需的土地、工具、资本和政治许可。一项发明若威胁行会、领主税权或法师专利，价值越高，发明者本人越可能先被招募、控制或夺走成果。

双陆世界的经济并不愚蠢。它能够跨大陆结算、给战争融资、为魔法风险定价，也能把一名没有身份的人精确估成若干金币。它最残酷的地方不是缺少市场，而是市场、身份和强制从来没有真正分开。
