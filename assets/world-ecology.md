# 魔物、生态与地下城

魔物不是等待冒险者触发的游戏刷新点。它们需要能量、食物、栖息地和繁殖条件，会争夺领地、逃避强敌、追随水源、学习道路规律，也会因为饥饿、疾病、迁徙和人类开发而改变行为。

## 什么会被称作魔物

“魔物”是猎魔、税法和冒险者任务使用的宽泛称呼，不是严格生物分类。

| 类别 | 特征 | 法律与处置 |
| --- | --- | --- |
| 普通野兽 | 无稳定魔力核心，行为主要由本能和环境决定 | 多按猎产、害兽或领主猎物管理 |
| 魔兽 | 身体和生态受魔力强化，可能有核心、元素器官或异常感知 | 依危险等级许可猎杀，材料需鉴定 |
| 智慧魔物 | 有语言、工具、交易、部落或长期计划 | 是否视作敌人、臣民或外国主体取决于法律和关系 |
| 人工生命 | 炼金、构装、召唤或实验制造 | 责任可能追溯到制造者；失控后按危险物处理 |
| 亡灵 | 由尸体、灵魂残留、诅咒或环境形成 | 神殿通常要求确认来源，避免破坏证据或扩散污染 |
| 神性污染生物 | 受残缺神力、旧神遗迹或权柄冲突扭曲 | 材料价值高但难保存，处理需专门许可 |

智慧并不自动意味着和平，外形可怕也不自动意味着无理性。国家会把影响征税和边境安全的群体称为魔物，也会与能够守路、交贡或提供材料的智慧族群签订事实条约。

## 魔物的繁殖、食物与领地

多数魔兽仍通过产卵、胎生、孢子、分裂或寄生繁殖。魔力会提高成活率、改变季节或让某些个体在能量充足时迅速成熟，却不能凭空取消食物。大型掠食魔兽数量必然少于草食兽、腐食者和小型猎物。

领地大小取决于体型、食物密度和移动方式。小型穴居魔兽可能在几平方公里内繁衍，成体飞行掠食者可能巡游数百公里。巢穴通常靠近水、食物、温度稳定区、魔力源或易守入口；发现幼体、粪便、啃痕和迁徙道，往往比直接看到成体更有情报价值。

魔物会回避经常巡逻的道路，也会记住商队休息点和牲畜气味。人类反复清剿可能把它们逼向邻村，而不是让风险消失。杀死顶级掠食者还可能造成中型魔兽过度繁殖。

## 等级与典型分布

等级反映生命层次与成长积累，不是物种固定职业。幼体、成体、首领、饥饿个体和受污染个体可以相差很大。

| 等级 | 典型分布 | 对定居社会的含义 |
| --- | --- | --- |
| 1—10级 | 农田边缘、城镇外圈、浅林、废屋、普通洞穴 | 猎人、民兵和低阶冒险者可处理；数量和偷袭仍能杀人 |
| 11—20级 | 边境森林、丘陵、矿道、商路空白和大型巢穴 | 需要成队猎人、装备与情报；可摧毁村庄或小商队 |
| 21—40级 | 深层荒野、地下河、魔力活跃区、海岸与高山 | 区域威胁，领主、公会和专业队伍必须介入 |
| 41—60级 | 大型地下城核心、海洋深层、污染区和古老领地 | 国家级资源与灾害；普通军队只能封锁、牵制或依靠器械 |
| 61级以上 | 极少数远古个体、灾区霸主、神性污染体与裂隙领主 | 足以改变区域生态和政治，通常需要多方情报、封印与强者合作 |

城镇周边并非只有低级魔物。高阶个体可能因迁徙、追猎、走私运输、法阵事故或伤病突然进入低风险区；反过来，深层荒野也充满低级猎物和寄生生物，不会只按区域整齐分级。

## 七类生态区域

### 城镇周边

农田、垃圾、牲畜和粮仓为鼠群、食腐兽、寄生孢子与小型掠食魔兽提供稳定食物。守卫和猎人压低高阶个体，却会让擅长躲藏、夜行和利用建筑的种类受益。

### 深层荒野

道路稀少，魔物领地完整。大型草食魔兽、群居掠食者、智慧部落和季节迁徙相互影响。远离人类不代表到处危险；真正风险常集中在水源、峡谷、繁殖地和迁徙交叉点。

### 地下

食物来自真菌、地下河、矿物魔力、地表渗入和捕食链。封闭环境让毒气、疾病、回声和坍塌比单个魔物更危险。矿区开挖会连接原本隔离的生态层。

### 海洋与内海

浅海魔兽与渔业共存，深海个体数量少但体型大。港口法阵能保护航道，不可能覆盖所有海域。大型海兽迁徙会改变航线、保险和沿岸粮价。

### 高山

低氧、寒冷、落石和狭窄通道限制数量。飞行魔兽、山羊类魔兽、晶矿寄生体和穴居群落占优势。矮人和山地国家常通过投食、声响标记或猎杀维持通道。

### 魔力灾区

魔力流量不稳定，普通物种也可能出现畸变。灾区边缘常有高价值材料和拾荒营地，核心区则可能随季节改变重力、温度、感知或繁殖规律。

### 魔界裂隙

裂隙周边的生态取决于泄漏物质、持续时间和是否存在智慧魔族控制。短暂裂隙造成污染和迷失，长期裂隙会形成异界真菌、寄生体、梦境干扰和本地物种适应带。不是所有裂隙都通向同一地点，也不是所有魔族都能随意通过。

## 地下城的七种形成方式

1. **古代遗迹**：宫殿、堡垒、观测站或仓库被掩埋；危险来自结构、守卫构装和后来的占据者。
2. **废弃矿区**：矿道连接地下河、气囊和巢穴；旧支撑和矿主隐瞒的事故记录至关重要。
3. **魔力结晶区**：能量富集形成晶体、异常植物和争夺核心的魔兽。
4. **空间折叠**：内部距离与外观不一致，可能由古代工程、灾害或失败传送产生；折叠仍有边界和维护逻辑。
5. **魔界裂隙**：异界物质和生物进入，本地生态逐步改变。
6. **神殿遗址**：权柄残留、祭祀机关、封存遗物和亡者誓约共同构成风险。
7. **人工试验场**：法师塔、军方或行会为了实验、训练和保密建造，废弃后责任记录常被销毁。

同一地下城可以多次叠加：矿山挖穿神殿遗址，法师又在其中建立试验场，最后裂隙吸引魔物占据。调查年代和建设目的，比给它贴一个单一名称更重要。

## 地下城不会自动刷新

宝箱是某人存放、遗弃、隐藏或来不及带走的物资，不会定期凭空生成。守卫被杀后不会按时间重生，除非存在繁殖群、召唤装置、亡灵循环或外部势力重新占据。矿脉和药材采完会枯竭，尸体会腐烂或被食腐者利用。

清剿后的地下城可能经历：

- 领主封锁、测绘、拆除机关与开采；
- 走私者、逃犯或新魔物迁入；
- 原有生态失衡导致别的物种扩张；
- 深层通道未查明，风险重新从内部出现；
- 商人夸大“已清空”或“仍危险”来影响地价和许可。

因此“完成一次任务”只能改变具体巢穴的现状，不能把整个地区永久标记为安全。

## 材料与经济价值

魔物材料的价值来自可验证用途，不是等级本身。常见商品包括皮甲材料、骨角、毒腺、腺体油、魔力核心、导魔组织、耐热鳞片、净化药材、孢子、晶矿伴生物和神性污染样本。

价格取决于：

- 完整度、污染程度、死亡时间和保存方式；
- 鉴定是否可靠，是否混入相似但有毒的部位；
- 法师塔、神殿、军方或工坊的当期需求；
- 采集许可、领主猎权和公会抽成；
- 运输重量、腐败速度和走私风险。

魔力核心不等于通用电池。不同结构、元素和污染会限制用途，错误切割可能爆裂或失效。高等级尸体很值钱，也可能重到无法运走、吸引其他掠食者或被国家宣布征用。

## 兽潮、迁徙与生态灾害

兽潮通常有原因：干旱、火灾、繁殖成功、顶级掠食者驱赶、地震、魔力潮、裂隙泄漏、人类围猎或有人故意投放诱引物。调查原因决定对策。只在城墙下杀死前锋，可能让后续群体转向更脆弱的村庄。

生态崩溃包括猎物过度减少、污染进入水系、寄生虫跨物种传播、巢穴密度过高和人工生命逃逸。魔物疾病既可能传给牲畜，也可能通过材料市场扩散。领主会在封路、隐瞒和保护税收之间摇摆；商人可能抢购药材，也可能继续出售受污染货物。

## 谁处理风险与收益

- **冒险者**接受侦察、护送、猎杀和采集任务，承担个人风险；
- **地方公会**登记任务、核验材料、发布危险级别并收取抽成，但信息可能滞后；
- **猎魔会**处理跨地区、伪装、附体和魔界相关威胁，不负责所有普通害兽；
- **领主**拥有封路、征发、猎权和土地处置权，也会压低赔偿、掩盖失职；
- **神殿**负责污染、亡灵、疫病和救济，优先级受教义与捐赠影响；
- **商人和工坊**提供预付款、运输与收购，可能垄断价格或推动过度猎杀。

任务发布者、出钱者、土地所有者和材料买家经常不是同一个人。RP中应明确谁承担清理费用、谁取得尸体、谁对误伤和后续污染负责。

## 可直接用于RP的标准魔物案例

以下是宏观标准案例，不覆盖已经进入正文的具体魔物、巢穴或清剿历史。

| 名称 | 常见等级 | 生态与行为 | 资源／风险 |
| --- | ---: | --- | --- |
| 灰袋鼠兽 | 2—5 | 城镇垃圾和粮仓边缘群居，受惊会释放恶臭粉尘 | 皮脂可制廉价防水膏；传播肠病 |
| 针背田豕 | 4—9 | 夜间翻掘块茎，成体护幼猛烈冲撞 | 肉可食、背针可制箭；毁田严重 |
| 灯苔螈 | 3—8 | 洞穴湿壁摄食真菌，以微光吸引小虫 | 发光腺用于矿灯药剂；干燥即失活 |
| 裂吻风犬 | 8—14 | 三至九只巡猎草原，利用侧风包围猎物 | 听骨用于侦测法具；会追逐马队 |
| 泥冠伏鳄 | 12—19 | 湿地浅水伏击，旱季集中在残留水塘 | 鳞皮耐水；尸体处理不当污染水源 |
| 铜翼食矿蛾 | 10—18 | 幼虫啃食低品位矿和金属构件，成虫迁飞繁殖 | 粉末可助冶炼；会破坏桥梁铆钉 |
| 镜眼林鹿 | 14—22 | 林缘草食魔兽，群体用反光眼斑迷惑追猎者 | 角用于幻术媒介；过度猎杀破坏林地更新 |
| 岩囊负重兽 | 17—27 | 高山杂食，吞石形成压舱囊，性情通常温顺 | 可驯作山地驮兽；惊群会引发落石 |
| 吞声蛛群 | 21—32 | 地下结网吸收振动，巢区会异常安静 | 丝用于隔音与侦测；无声区使队伍失联 |
| 盐雾鳐 | 24—36 | 内海近岸迁徙，借盐雾滑翔捕食鱼群 | 膜翼用于防腐法具；会撞翻小船 |
| 赤髓藤群 | 28—40 | 污染区植物群体，以尸体矿物和魔力生长 | 髓液是止血原料；切割会释放诱捕孢子 |
| 旧钟守像 | 32—48 | 古代人工守卫，按残缺的时间表巡逻而非觅食 | 核心齿轮和符片昂贵；错误口令会扩大警戒区 |
| 深潮甲鲸 | 41—58 | 深海滤食巨兽，繁殖季靠近暖流 | 骨油和甲板价值高；死亡搁浅会引发争夺与腐败灾害 |
| 断祷亡骑 | 38—55 | 神殿遗址中由未完成誓约维持的亡灵小队 | 装备可能保留祝圣；直接破坏会释放诅咒 |
| 空壳巡游者 | 48—63 | 空间折叠区的半实体捕食者，沿固定几何路径出现 | 膜片可研究空间术；越界追击会失去方向 |
| 黑曜脊母 | 61—72 | 火山高原的古老群体首领，控制多个低阶巢群 | 甲壳是国家级材料；击杀可能触发幼群外迁 |

案例等级是常见区间，不是固定数值。幼体、首领、饥饿、受伤、驯化和污染都应改变判断。

## RP执行口径

出现魔物前先回答：它吃什么、为何在这里、巢穴和幼体在哪里、当地人怎样适应、杀死后谁获利。地下城要有形成原因、历史层次和资源消耗，不得自动刷新。

任务结算后还要考虑尸体、污染、权属、逃散个体和生态空位。一次胜利可以真实改善局部安全，也可能制造下一轮更隐蔽的问题。
