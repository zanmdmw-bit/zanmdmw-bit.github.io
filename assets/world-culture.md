# 文化、语言与历法

大陆上的人能够跨国做生意，并不意味着他们在家里说同一种话。商路、神殿和公会制造了共享工具，村庄、家族与族群则保留自己的口音、记忆和文字。语言既是交流能力，也是身份、合同风险和地方权力。

## 大陆通用语不是共同母语

大陆不存在所有人的共同母语。五大帝国的宫廷语言彼此不同，十七王国与公国拥有连续变化的地方语群，精灵、矮人、兽人、魔族和地方族群也各有母语、方言与部分独立文字传统。同一国家内部，边境农民和王都官员甚至可能需要通用语才能完整沟通。

**大陆通用语**是一门后天学习的跨族群语言。它在黑日战争的联军、神殿补给、公衡币制和跨国商路中逐渐固定，今天主要用于：

- 跨国贸易、外交和多国军议；
- 神殿、公会、冒险者登记与交通节点；
- 港口、关卡、传送院、空港和大型集市；
- 不同种族、不同母语群体之间的最低限度交流。

通用语不是多数本地人的母语。本地人在家庭、村庄、同族群、地方行会和宗族仪式中主要使用自己的母语。会说通用语也不等于说得标准：商路口音够用来议价，却未必能理解法庭反问；军营口令能协同行动，却未必能谈抽象学术。

普及程度随地点和身份变化：

| 场景 | 能进行日常通用语交流者的常见比例 |
| --- | ---: |
| 帝都、王都核心城区 | 55%—80% |
| 大港、国际商路、神殿和公会节点 | 60%—90% |
| 普通地方城市 | 35%—65% |
| 城镇与繁忙驿路村 | 20%—45% |
| 普通农业村庄 | 5%—25% |
| 偏远山村、封闭族群、深层矿居 | 0%—15% |

这些比例不是学校统计，而是RP判断尺度。一个村庄可能只有村长、退伍兵、行商和神殿侍从会说；他们会自然成为外来者的翻译、担保人和信息过滤者。

## 母语体系与语言政治

人类语言大致形成中央诸语、西陆瓦尔语群、东环米尔语群、南海塞勒语群、北陆诺德语群、南方萨法语群和东南林缘语群。国境不等于语言边界：战争、移民和联姻会让同一语群跨越数国，也会让一个王国内存在三四种地方话。

永翠精灵语在树城、林地和德鲁伊仪式中存在年龄、礼仪与自然对象的细密区别；高山矮人语重视矿脉、工序和契约证词，三个王国的符文拼写并不完全相同；兽人语言随草原、山堡、森林和火山传统分化，部族手势与战旗能补足跨方言口令；魔界诸语更分散，契约语、梦境语和裂隙黑话不能视作一门统一“魔族语”。

统治者会出于成本而使用通用语，也会刻意扶持宫廷母语。任命、税册和军令若只用中央语言，地方书记就能垄断解释；若承认多语文本，国家又必须承担翻译、版本冲突和伪造风险。语言政策因此常是财政和权力问题，不是单纯文化爱好。

## 通用文字、地方文字与专业符号

大陆通用文字是通用语的跨国书写规范，常用于商会契约、公衡账簿、外交副本、冒险者登记和交通告示。它有印刷体、书记体与快速记账体；认识其中一种不保证能读懂另外两种潦草写法。

地方文字记录本地语言、地名、族谱、习惯法和宗教传统。某些人类语言借用通用字母，另一些仍使用本国音节字、贵族纹章速记或神殿抄写传统。精灵树纹、矮人刻符和兽人结绳—刻痕系统都能精确记录信息，却要求不同的阅读习惯。

古代文字是对失落政权、黑日战争以前文本和遗迹铭文的统称，不是一门万能古语。同一地下城的门铭、维修册和祭祀壁画可能来自三个时代。古文字学者必须判断年代、书写材料和抄写错误。

魔法符号不是普通文字的华丽版本。它记录魔力方向、结构关系、触发条件和安全边界；同一符号在不同学派中可能含义相反。会读法书的叙述文字，不代表能安全补全法阵。军用暗语、神术礼仪语、契约缩写和炼金标记也各有训练门槛。

RP中，阅读能力必须按“语言＋文字体系＋专业领域”判断。会读通用账单的人，未必看得懂地方地契；会读地方诗歌的人，未必能识别军需缩写。

## 识字率与教育渠道

稳定文明区的整体功能性识字率约为15%—28%，但差异极大。帝都、王都和商贸城可达35%—60%；普通城镇约15%—35%；农业乡村常在3%—15%；偏远地区可能由一名书记、神官或退伍军需员替整个村落读写。

“识字”分层明显：

- 能认姓名、数字、价签和常见告示；
- 能写短信、收据和简单账目；
- 能阅读契约、法律与复杂账簿；
- 能处理多语文本、古文、神术或魔法符号。

书记员因此是一种现实职业。地方书记替人写信、读契、登记婚丧、抄税册，既提供服务，也可能隐瞒条款、收取双份费用或替权贵制造“合法”版本。复杂交易通常要求双方书记、见证人和封印，正因为口头承诺与文字解释并不可靠。

教育渠道包括：

- **家庭与村社**：教授母语、农事、身份礼仪和少量数字；
- **神殿学校**：教授祷文、通用语、基础读写、抄写和慈善医护；学费可用服役抵偿；
- **城市私塾与官署学校**：培养书记、税吏、法庭助理和商号账房；
- **贵族家庭教育**：重视家谱、礼法、领地财政、骑术、外交语言和婚姻政治；质量取决于家族肯花多少钱；
- **行会学徒制**：把读写嵌入工序、账目、质量标准和师徒服从；行会不会免费培养潜在竞争者；
- **法师塔与学院**：要求较高识字、数学、符号与材料知识，学费和担保远高于普通学校。

学校不能自动创造现代大众教育。儿童劳动、路程、冬季、学费、书写材料、教师数量和领主许可都会限制入学。行会、神殿和贵族愿意教育需要的人，却没有理由主动消除自己的知识垄断。

## 穿越者的语言适应硬规则

地球穿越者完成生命模板重构后，会自动获得**大陆通用语的听说能力**。这是抵达后的生存适配，不是持续运行的翻译系统。

执行时必须遵守以下边界：

1. 自动获得的只有大陆通用语听说，不包括任何文字能力。
2. 穿越者可以和会通用语的本地人交谈；对方只会地方语言时，双方仍无法直接交流。
3. 穿越者起初看不懂路牌、契约、账簿、商品标签、公会任务板、登记文件、法律文书、法书和魔法符号。
4. 穿越者必须自行学习大陆通用文字及其他文字；学习速度取决于教师、练习、原有教育和时间。
5. 穿越者完整保留原本掌握的地球语言。中国人仍会中文；原本会英语、日语或其他语言者也保留对应能力。
6. 地球语言不会被自动翻译。本地人听见中文、英语或日语时只会听见陌生语言。
7. 穿越者不会自动掌握地方语言、精灵语、矮人语、兽人语言、魔界语言、古代语、神术语、军用暗语和魔法符号。
8. 系统不存在对新语言的实时翻译、字幕或文字识别功能。
9. 本地人和其他穿越者可以通过正常学习掌握地球语言。

因此，“穿越者完全无法与本地人口头交流”是错误口径；正确口径是：**他们能用通用语与会通用语者说话，但一开始是文盲，也无法跨过地方语言和专业文字壁垒。**

## 地球语言的情报价值

地球语言最初可以充当小群体暗语。穿越者可用中文讨论价格、计划或身份，前提是现场没有懂中文的其他穿越者、本地学者或已经接受培训的监听者。它不是永久密码。

商会、王室、法师塔和情报机构一旦确认地球语言有稳定规律，就会招募穿越者编词表、训练翻译、收集书写样本。最先失去保密性的通常是数字、姓名、常用命令、物品和粗口；技术术语与文化隐喻需要更久。

地球人内部也不存在共同语言。不同国家穿越者可能必须反过来使用大陆通用语交流。掌握多门地球语言的人会成为翻译、经纪人或被控制的情报资产，也可能因误译、偏袒和夸大能力引发严重后果。

## 公衡历与地方纪年

跨国契约、神殿档案和商路普遍使用**公衡历**。一年为365日：十二个月，每月30日，年末另有五个“界末日”。十二月按农业与天象常称为：融土、初芽、长雨、青麦、盛阳、长昼、熟禾、金穗、落叶、长风、初霜、深冬。地方名称可以不同，正式契约同时写数字月以防误解。

每月分上旬、中旬、下旬，每旬十日。旬末常是结账、集市、轮休或小型祭日，但领主庄园、矿山和军队不会因此统一停工。神殿还保留各自的七日礼仪循环，两套周期并存，偶尔造成“休息日属于谁”的争执。

跨国史书以黑日战争最终封印完成为共同纪年起点，称封印纪年。宫廷仍常用君主在位年、王朝年或地方灾年。严谨文书会并列两种日期；只写“国王十二年”的旧档案必须先确定是哪位国王。

季节不是全大陆同步模板。北境长冬、南方内海湿热、东南林区多雨，农业月份会按纬度、海拔和魔力异常调整。公衡历提供共同日期，不能保证同一天各地都在播种。

## 一天、钟楼与跨地区时间

公衡计时把一昼夜分为24时，一时60分；精密法阵和航海记录再把一分分为60息。普通人仍按晨祷、开市、正午、日落和夜更生活，不会随身携带精确时钟。

计时工具按成本分层：

- 日影、燃香、漏壶和沙漏用于家庭、作坊与岗哨；
- 城市钟楼按本地正午校准，误差可能累积到十几分钟；
- 港口、传送院和军务厅使用校准机械钟或低阶魔法计时器；
- 星环观测台和大型法阵拥有精确统一计时，但维护昂贵。

大陆东西跨度约48,000公里，各地日出和正午显著不同。日常约会默认使用当地钟楼时间；远程航运、传送预约和外交电文必须注明出发节点、到达节点和日期。商人说“第三日午前到港”，指目的港当地时间，不存在普通人随时可查的全大陆统一实时时钟。

## 节日、纪念日与农业节律

跨国最常见的公共纪念包括封印纪念日、公衡誓约日和各主神的大祭。它们的政治意义不同：封印纪念日强调共同灾难与军役，公衡誓约日由商会、神殿金库和官署重校秤砣，各神祭则争夺捐赠、巡游路线与城市空间。

地方节日更多围绕播种、雨季、收获、牲畜迁徙、开河、封港、矿炉点火和祖先祭祀。节庆不是纯粹休闲：领主展示慷慨，行会确认成员，青年寻找婚配，商人集中收债，盗贼利用拥挤，神殿统计信众。灾荒年取消宴席可能引发怨恨，强行举办又可能耗尽粮仓。

## 公衡单位与地方旧单位

跨国贸易使用公衡单位，站内叙述同时给出现代换算，便于RP保持距离和载重一致。

| 类别 | 公衡单位 | 约合现代单位 | 常见用途 |
| --- | --- | ---: | --- |
| 长度 | 1指 | 2厘米 | 布料、刃厚、零件 |
| 长度 | 1掌 | 10厘米 | 木工、器皿 |
| 长度 | 1步 | 1米 | 房屋、绳索、短距离 |
| 长度 | 1里 | 1公里 | 道路和地图 |
| 路程 | 1驿程 | 约30公里 | 商队日程，不是精密长度 |
| 重量 | 1衡两 | 50克 | 药材、贵金属 |
| 重量 | 1衡磅 | 0.5公斤 | 食物、货物 |
| 重量 | 1石 | 50公斤 | 粮袋、车载大宗货物 |
| 容量 | 1升 | 约1升 | 油、酒、谷物 |
| 容量 | 1斗 | 10升 | 粮税和仓储 |
| 容量 | 1桶 | 约100升 | 船运、酒和淡水；实物桶须验容 |
| 面积 | 1亩 | 1000平方米 | 城郊土地与小农田 |
| 面积 | 1顷 | 100亩／0.1平方公里 | 庄园、林地和大田 |
| 时间 | 1时 | 60分 | 官署、交通与法阵 |

地方旧单位仍广泛存在。“一车”“一袋”“一井地”“一日犁程”会随牲畜、土质和习惯变化。公衡院只在跨国交易、纳税总册、军需和受理争议时要求换算；地方领主、行会和商人可能故意保留模糊旧单位，从换算误差中获利。

交易中必须区分标称与实测。粮食按容量卖时受含水量影响，矿石按重量卖时受杂质影响，土地按面积卖时又可能不包含水权、林权和道路。公衡单位减少争执，却不能消灭欺诈、腐败和权力差。

## RP执行口径

每次跨地区交流先判断双方母语、通用语程度、识字范围与专业术语。文盲可以聪明，贵族可以会多门口语却读不懂魔法符号，书记可以借信息差操纵契约。日期要说明使用何地时间，路程要区分直线、道路和驿程，商品要说明采用公衡还是地方旧单位。

文化设定的作用不是给所有场景增加翻译障碍，而是在关键节点让语言真实影响担保、误会、价格、情报和权力。
