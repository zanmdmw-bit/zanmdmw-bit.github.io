等级是世界法则对生命状态的简化记录，不是职业证书，也不是决定胜负的血条。它把身体、灵魂、魔力与已经完成的生命蜕变压缩成一组可读数字，却不会替一个人学会剑术、法术、战术和勇气。

所以，一名十级士兵可能凭借队形、长枪和同伴杀死更高等级的野兽；一名属性惊人的贵族也可能因为从未见过真正战场，在第一次伏击中死去。等级能够说明生命抵达了什么层次，不能脱离现实条件单独说明“谁一定赢”。

## 等级不是职业

本世界不存在战士、盗贼、法师职业栏，也没有转职任务、职业专属装备、固定技能树和强制的坦克、输出、治疗分工。“骑士”“刺客”“猎人”“法师”只是训练方向、谋生方式或社会身份。

盗贼可以穿甲持盾，法师可以学习弓术，骑士也能研究治疗术。能否真正使用，取决于有没有老师与知识、身体是否适应、训练是否足够、装备是否适合当前环境。等级提升不会自动赠送技能，面板也不限制武器。

普通本地人看不见别人的精确等级。公会记录、神殿仪式、魔法检测、鉴定道具、讨伐证明、声望和实战表现只能提供估计，因此谎报、隐藏和误判实力都很常见。

## 面板记录什么，又遗漏什么

面板综合反映身体结构、神经反应、灵魂强度、魔力容量、感知能力、生命本质与已经完成的蜕变，现行基础属性共有六项：

| 属性 | 主要代表 | 不会自动提供 |
| --- | --- | --- |
| 力量 | 爆发、挥击、负重、擒抱、投掷与结构破坏 | 命中、速度、战术与法术抗性 |
| 敏捷 | 神经反应、协调、动作转换、短距爆发与武器控制 | 视野、移动空间与对未知攻击的自动闪避 |
| 体质 | 耐力、恢复、抗冲击、抗病、器官与失血承受 | 不死、灵魂防御与对同级致命伤的免疫 |
| 精神 | 注意力、意志稳定、施法控制、灵魂承载与恢复 | 知识、智谋、情商与绝对精神免疫 |
| 魔力 | 魔力容量、输出上限、恢复与法术承载 | 法术知识、咏唱速度和控制精度 |
| 感知 | 视听、危险直觉、追踪、距离与魔力波动判断 | 全知、识破完美伪装与神明级遮蔽 |

面板不会显示剑术熟练度、战术智慧、法术知识、社会地位、装备质量、实战经验、勇气和临场判断。这些被遗漏的东西，往往正是同等级者之间决定生死的部分。

## 属性与升级

普通健康成年人的单项属性通常为 **8—12点**，10点可视作平均水平。每次升级，全属性自动增加1点，并获得5点可自由分配的属性；自由点只能强化基础，不能直接购买知识或技能。

| 属性值 | 大致生命层次 |
| --- | --- |
| 10 | 普通成年人 |
| 15 | 训练良好的士兵或猎人 |
| 20 | 普通人类自然极限附近 |
| 30 | 明显超出凡人 |
| 50 | 低阶超凡身体 |
| 100 | 对普通人形成绝对属性压制 |
| 200 | 英雄级怪物或顶级强者 |
| 500以上 | 传奇、神话或神权核心领域 |

在普通人范围内，1点属性大约造成8%—12%的实际差异；进入高属性后不能继续按简单百分比叠加，因为增长已经涉及骨骼、肌肉与魔力结合、神经传导、灵魂控制和生命结构的改变。同样是力量100，人类、巨人、古龙、史莱姆和昆虫型魔物的表现也不会相同：属性说明能力，身体结构决定能力怎样释放。

## 经验从哪里来

凡人阶段最稳定的经验来自真实击杀。目标基础经验约为 **目标等级²×10**，再按性质修正：强壮或特殊个体约为1.5—2倍，精英通常3倍，首领5—10倍，区域霸主10倍以上，特殊神话目标单独计算。

世界法则会判断谁造成伤害、谁限制目标、谁提供关键控制，以及谁真正构成了死亡结果。经验不是简单平均分队；站在旁边、摸一下尸体、对必死目标补刀或跟随高等级者观看，都不能稳定获得经验。

高等级者屠杀低等级目标会严重衰减。目标低6—10级时经验已明显下降，低11—20级只剩少量，低20级以上接近于零。因此屠杀平民、家畜和低级怪物不能让传奇强者持续成长；这种行为可能有军事与经济结果，却几乎没有修炼价值。

## 升级公式与阶段瓶颈

从当前等级升到下一级，所需经验为 **当前等级²×100**。

| 当前等级 | 升级所需经验 |
| ---: | ---: |
| 1 | 100 |
| 5 | 2,500 |
| 10 | 10,000 |
| 11 | 12,100 |
| 20 | 40,000 |
| 40 | 160,000 |
| 60 | 360,000 |
| 80 | 640,000 |
| 89 | 792,100 |
| 99 | 980,100 |

经验达到要求后，同一阶段内可以直接升级；跨越20、40、60、80与90级时，必须完成对应的生命突破。经验只能把人推到门前，不能替他越过门槛。

## 六个生命阶段

**1—20级为凡人阶。**大多数士兵、冒险者、平民和常见魔物都在这里，仍可被数量、弩箭、毒药和陷阱杀死。20升21级必须凝聚超凡核心。

**21—40级为超凡阶。**生命本质开始脱离普通人，可以主动吸收世界源质，恢复、魔力与寿命明显提高；普通军队仍可通过重弩、围攻、毒药和法阵处理。40升41级必须形成代表个人道路的英雄印记，例如剑、火焰、守护、疾风、狩猎、梦境、血脉或神术。

**41—60级为英雄阶。**个人足以影响大型战役、破坏普通城墙和军阵，形成明确战场风格；多数王国的最强者处于此段。60升61级必须建立传奇领域，把自身道路稳定投射到周围世界，领域不一定表现为肉眼可见的结界。

**61—80级为传奇阶。**单人能够改变地区战争，对普通军队近似灾害，寿命常达数百年并拥有稳定领域；五大帝国的镇国强者主要在这一阶段。80升81级必须完成神话真身。

**81—89级为神话阶。**生命接近凡世正常容纳上限，可以造成大范围地形或气候改变，每一个公开个体都有历史记录，普通国家无法独自处理。古龙、顶级魔王和极少数古代强者属于这里。

**90—99级为神祇阶。**90级生命有资格点燃神火、凝聚神格、取得权柄、越过天涯环带并进入神庭，但不会自动成为神。完整神职、权柄碎片、主神承认、空缺神位、稳定信仰或自凝神格至少需要满足一种；没有权柄的90级生命称为准神。

**100级为主神阶。**这是当前世界面板能够完整记录的最高层次。抵达这里不能只靠经验，必须完整掌握主要世界权柄、获得法则承认、建立稳定神格和神座。九位主神都在100级，核心属性通常可能超过500，但真正可怕的是权柄能够改变规则，而不是某一项数字更高。

## 高阶成长不只依靠杀戮

普通跑步、挥剑、读书和冥想不直接增加面板经验，只会增加技巧、理解、身体控制与突破成功率。完成超凡核心突破后，生命才可以通过真正改变本质的修炼缓慢获得经验，包括本源淬炼、压缩魔力、重构精神模型、吸收世界源质、完善血脉、建立领域和感悟权柄。

高品质魔晶、地脉精华、古代药剂、魔兽核心、神性材料与魔界源质能够加速成长，也可能令承载不足者中毒、爆体、魔力畸变、血脉污染或精神崩溃。大型地脉、元素风暴区、世界边缘、古代神殿、龙脉、魔界裂隙、深层地下城和神性遗迹都是修炼地，也是死亡率极高的灾区。

龙族、精灵、魔族、兽族与特殊人类可以通过血脉觉醒获得经验和属性改变，但通常需要年龄、资源、祖先遗物、生死刺激、仪式或同族引导。神明赐福可以提供经验、神力、突破资格、神术和权柄碎片，代价则可能是信仰、契约、代理职责、自由和继承神明的敌人。

英雄阶以上完成足以改变世界的功业——封印大型魔界入口、阻止灭国灾害、修复世界级地脉、击退古神投影或保护数百万人——也可能得到法则反馈。这不是任务奖励，而是行为本身在世界中留下了不可忽略的印记。

## 时间、寿命与强者稀缺

没有资源与机缘时，21—40级纯修炼升一级常需数年到数十年，41—60级需十年至上百年，61—80级需数十年至数百年，81—89级甚至需要百年至上千年；90级以上不可能仅靠普通修炼。

长生种可以用数百年积累，人类若没有延寿药剂、血脉改变、神术、高阶突破或特殊传承，通常活不到自然修炼至传奇的年龄。延寿也不等于升级：秘药能给君主更多执政和训练时间，却不能凭空提供击杀经验、英雄印记与传奇领域。

大陆、魔界与已知神庭范围内，21—40级约有15万—25万人，41—60级约3,500—5,000人，61—80级约140—200人；81—89级已确认21名，推测不超过40名；神庭承认的90—99级从神与小神约二十余名，100级主神只有九位。完全未知的封印存在不计入统计。

## 高等级怪物为什么仍然存在

文明稳定控制区只占大陆约四成，余下的无人荒野、深层地下城、古代遗迹、魔力灾区、龙族领地、世界边缘、海底深渊、魔界和封印生态区仍有高阶目标。1—10级怪物在城镇附近常见，11—20级多见于危险荒野与小型巢穴，21—40级是深层荒野和地区霸主，41—60级已是区域灾害，61—80级包括古代巨兽、成年龙和高阶魔王眷属，81—89级则是能够影响大陆政治的世界级独立存在。

高级怪物不会像低级魔物一样不断刷新。41级以上通常拥有固定领地与生态地位，61级以上的名字、习性和疆域会被国家长期记录。它们数量稀少、有情报、有族群与盟友，击杀风险接近同归于尽；其材料还会引发国家和组织争夺。因此高阶强者通常是修炼数十年、寻找一次机缘、完成一次突破、冒险击杀一两个同级目标，再继续漫长修炼，而不是每天进森林寻找同级怪物。

## 现实战斗怎样判断

完整战斗力至少由九部分组成：等级、属性分配、种族与身体结构、武器装备、技巧训练、法术战技与血脉能力、距离地形与机动、情报与克制，以及当时的伤势、疲劳和心理状态。

力量高者可能被飞行和远程攻击持续拉开距离；敏捷高者仍可能死于背后弩箭、箭雨、无死角爆炸与狭窄通道；高体质者仍会被斩首、穿脑、毁心、灵魂攻击与同级毒药杀死；高精神者可能被政治欺骗；高魔力者若不会法术，只是拥有大量未被利用的能源；高感知者也无法发现完全陌生且没有外泄的诅咒。

等级相差5级以内，技巧、装备、环境和克制足以改变胜负；相差5—10级，低等级者通常需要偷袭、人数、强力装备、针对性毒药、地形或对手负伤；相差10—20级，正面战斗极不利，但暗杀、封印、陷阱和神器仍可能成功；相差20级以上，普通正面对抗基本没有胜算，只有契约漏洞、神性毒药、世界级法阵和同级神器等手段可能绕开直接属性压制。

## 初代勇者与神庭门槛

初代勇者阿尔德雷恩在黑日战争最终阶段达到 **90级**。他在深渊王门前点燃神火，理论上已经有资格进入神庭成为神祇，却把神火、神格雏形、大部分力量和自身存在投入九重界锁，没有建立神座。

因此他可能已经死亡，也可能以不完整神祇状态与封印融为一体。九位主神不能轻易收回他的神格，因为那可能令深渊王门重新开启。阿尔德雷恩是极少数已经越过凡世上限，却没有成为神庭正式神祇的存在；他的90级也再次说明，凡人史诗的顶点仍低于100级主神。

## 千叶为何是规则中的特殊变量

千叶遵守同一套等级、经验、属性与突破规则。他不会因为拥有系统就自动越级，也必须真实训练、完成击杀、承受身体负荷并学习如何控制力量。真正异常的是，他取得成长条件的速度和范围远超本土生命。

异世界生存系统能够提供中文解释、跨体系兼容知识、精确训练步骤、能力需求检测、进度提示、系统装备、恢复与生存资源、其他文明的武器物品，以及本世界从未出现过的完整超凡体系。本地人需要多年寻找老师、法书、材料和试验场，他可以用生存点购买已经翻译并排好训练顺序的知识；残次材料还可通过系统回收，转化为新的资源。

他的金色剑体、巨刃显化、高速剑术与魔力强化组合，可能被本地人误判为圣剑传承、古代勇者体系、神术、高阶魔剑士、异界皇室秘术、特殊血脉或神器能力。实际来源可能不属于本土任何咏唱语言、元素分类、法术模型和学派理论。

其他地球人拥有个人面板，却没有千叶的系统商城、生存点、回收、限时折扣、跨世界搜索、知识窗口和兼容训练包。系统本身是地球迁移工程之外的独立跨界生存终端，其创造者与绑定原因尚不确定，普通神术也无法看穿完整本质。

系统仍受生存点、购买价格、学习时间、身体承载、能量消耗、现实风险、秘密暴露与使用后果限制。千叶可以力量很高却被远程风筝，可以敏捷很高却踩中未知魔法陷阱，也可能因为精神、感知或魔力不足而无法发挥一项异界能力。**他不是不受规则约束的人，而是拥有一条能不断把其他世界资源引入现有规则的私人入口。**
